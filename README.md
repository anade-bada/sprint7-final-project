# 📱 Análisis de Telecomunicaciones - ConnectaTel

## 🎯 Objetivo del Proyecto
Analizar patrones de uso de servicios móviles (llamadas y mensajes) de ConnectaTel para identificar segmentos de clientes, detectar comportamientos atípicos y optimizar la oferta comercial en México y Colombia.

## 📊 Datasets Utilizados
- **`plans.csv`**: Catálogo de planes (precios, minutos incluidos, GB, costos extra)
- **`users_latam.csv`**: Información de clientes (edad, ciudad, plan contratado, fecha registro)
- **`usage.csv`**: Detalle de uso real (duración llamadas, longitud mensajes)

## 🔄 Etapas del Análisis
1. **Exploración inicial**: Carga y estructura de datos
2. **Calidad de datos**: Detección de valores nulos, sentinels y fechas inconsistentes
3. **Limpieza**: Tratamiento de valores atípicos y estandarización
4. **Estadísticas descriptivas**: Medidas centrales y dispersión por segmentos
5. **Visualización**: Histogramas, boxplots y detección de outliers
6. **Segmentación**: Clasificación por edad, uso y comportamiento
7. **Insights ejecutivos**: Conclusiones y recomendaciones comerciales
